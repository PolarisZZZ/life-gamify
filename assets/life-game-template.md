### 📊 记录规则
- 数据来源：`body-metrics.md`
- 每日称重 + 体脂记录（早起、空腹、同一时间段）
- 以 **7日移动平均** 作为判定（减少日波动影响）

### 🎯 奖惩规则（每周结算一次）
**体重（以7日均值对比上周）：**
- ↓ 0.3–0.6 kg：+30 经验 / +30 分数（理想区间）
- ↓ 0.6–1.0 kg：+40 经验 / +40 分数（偏快）
- ↑ ≥0.3 kg：-20 经验 / -20 分数
- 变化在 ±0.3 kg 内：+10 经验 / +10 分数（维持稳定）

**体脂（以7日均值对比上周）：**
- ↓ ≥0.3%：+20 经验 / +20 分数
- ↑ ≥0.3%：-20 经验 / -20 分数

**打卡奖励：**
- 每日完整记录（体重+体脂）：+5 经验 / +5 分数

**连续签到奖励：**
- 7 天连续：+35 经验 / +35 分数
- 14 天连续：+70 经验 / +70 分数
- 30 天连续：+150 经验 / +150 分数

## 📌 规则说明
1. 完成任务 → 获得对应分数的**经验值**（累加制，不清零）
2. 完成任务 → 同时获得**同等数量的奖励分数**
3. 经验值达到阈值 → 自动升级，额外获得**升级奖励分数**
4. 奖励分数可兑换自定义奖励（消费时告诉我）
5. 标记"可重复"的任务可以多次完成
6. **分数匹配机制**：由系统根据任务难度与重要性初步提议分数，由用户确认后生效
